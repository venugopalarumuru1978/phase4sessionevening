1. BootStrap
----------------
What is a BootStrap ?
	It is a CSS Framework  is used to develop responsive design based web pages. 

What is a Responsive Web Design? 
	It is a designing which is suitable for different gadgets for data presentation. 
		i.e.  this design makes adjust the content based on device screen size. 
Html Versions :- 

Latest version of HTML is HTML5. 

Html5 and CSS3 version having responsive web design features. 
With the help of HTML5 and CSS3  features we got a css framework called as BootStrap.


We can download bootstrap framework from "www.getbootstrap.com"
-----------------------------------------------------
Steps to configure Html page for BootStrap :- 

1.  Add doctype in html page at the top of program.
		<!DOCTYPE html>

2.  Set the Universal Character Set in web page by using <meta> tag(<head>) 
 		<meta charset='utf-8'> 
3.  Set ViewPort to identify  device width while loading page, it has to be done with <meta> tag.
	<meta name='viewport' content='width=device-width, initial-scale=1'>
			initial-scale=1 :-  Mobile First, it verifies first mobile screen size
				if it found content adjust as per the width otherwise it checks other 				devices for width calculation

4.  Add "bootstrap.min.css" and "bootstrap.min.js" file into webpage. 
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0-alpha2/dist/css/bootstrap.min.css" rel="stylesheet">

<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0-alpha2/dist/js/bootstrap.bundle.min.js"> </script>
